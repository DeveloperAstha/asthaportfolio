#### Preview

#### Download
 - [Download from ThemeWagon](https://themewagon.com/themes/resume-nextjs/)


## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
